# Blood Bowl Team Manager - Style Guide

## Color Palette

- **Primary Blue**: `#1d3860` - Main brand color, used for headers, buttons, borders
- **Primary Red**: `#922d26` - Accent color for danger actions, hero sections
- **Gold**: `#eaaa02` - Highlight color for navigation, titles
- **Parchment**: `#fffef0` - Background for paper-like effect
- **Light Blue**: `#e0f0ff` - Table backgrounds
- **Hover Blue**: `#d4e8ff` - Interactive hover states
- **Dark Blue**: `#092540` - Button hover states
- **Dark Red**: `#701d1a` - Danger button hover states

## Typography

- **Base Font Size**: 16px (1rem)
- **Hero Titles**: 3.5rem, bold, italic, gold color
- **Section Titles**: 2rem, blue color
- **Body Text**: 1rem, line-height 1.6

## Components Overview

### Layout Components

- **AppBar** - Top navigation with logo and auth links
- **Hero** - Angled header with SVG background
- **Footer** - Bottom page section
- **Parchment** - Paper-textured background overlay
- **ContentContainer** - Main content wrapper
- **MinHeightContainer** - Ensures minimum viewport height

### UI Elements

- **Button** - Primary action button (blue)
- **DangerButton** - Destructive actions (red)
- **RemoveButton** - Compact delete button
- **InlineLink** - Link styled as button
- **Toggle** - Switch component with blue/red states
- **Select** - Custom dropdown with arrow icon

### Content Components

- **TextBorderBox** - Bordered content with drop cap
- **Table** - Striped data table with blue theme
- **Titles** - H1, H2, and subtitle variants

### Feature Components

- **TeamBuilder** - Multi-step team creation
- **LeagueForm** - League management
- **MyTeamsList** / **MyLeaguesList** - User content lists

## Design Patterns

- **Borders**: 2-3px solid borders using primary blue
- **Hover Effects**: Subtle background color transitions
- **Disabled States**: Light gray with reduced opacity
- **Spacing**: 10-20px margins/padding for consistency
- **Border Radius**: 4px for buttons, 34px for toggle switches
